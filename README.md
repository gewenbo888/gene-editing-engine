# Gene Editing Engine · 基因编辑引擎

**The Nature of Genetic Engineering, Biological Programming, Evolutionary Control, and Post-Human Civilization**

A civilization-scale, fully bilingual (English · 中文) atlas of gene editing — DNA as code, CRISPR as a word processor, gene therapy as a one-time edit, synthetic biology as authored life, AI as the new prior, and the deep question of what happens when a species becomes able to edit itself. Ten interactive systems span the four-letter alphabet of DNA, the four-billion-year edit by natural selection, the rise of CRISPR, the cures already on the market, the therapy→enhancement ladder, synthetic-biology circuits, the AI×biology stack, planetary bioengineering, post-human society, and a unified Programmable-Life model, closing on a recursive engine that runs the same principle — *information that knows how to copy itself* — from atom to post-biological civilization.

Every diagram, simulation, and dashboard switches language live, with no reload.

## Visualizations

- **GenomeField** — hero canvas of a rotating DNA helix + drifting A·T·G·C glyphs + edit events
- **EvolutionSim** — 80-organism selection canvas + 6 forces + 4 modes of steering
- **CRISPRLab** — flagship animated CRISPR diagram (gRNA → Cas9 → cut → repair → verify) + 5 tools + off-target scan
- **TherapyMap** — 9 gene therapies across 4 regulatory tiers + delivery-method schematics + approval timeline
- **EnhancementLadder** — 6-rung therapy→enhancement SVG ladder + ethical-pressure gauge
- **SynBioCircuit** — 6 genetic logic gates + 5-milestone synbio timeline (Repressilator → Sc2.0)
- **AIBioStack** — 5-layer Read → Predict → Design → Plan → Test stack + folding-protein animation
- **EcoEngineering** — world-map of 6 planetary projects + irreversibility halos + containment axis
- **PostHumanSociety** — 8-axis grouped bars across today/enhanced/designed + 3 access regimes
- **ProgrammableLifeModel** — interactive 8-axis SVG octagon radar across natural / edited / synthetic profiles
- **BioRecursion** — 11-substrate ladder, atom → post-biological civilization, with keyboard nav + auto-play
